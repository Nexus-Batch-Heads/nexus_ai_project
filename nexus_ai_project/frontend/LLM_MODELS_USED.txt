LLMs Used in Nexus AI
=========================

1. GPT-4o Mini (OpenAI)
   - Model ID: gpt-4o-mini
   - Used when OPENAI_API_KEY is set
   - Powers: Decision simulations and AI chat

2. LLaMA 3.1 70B Versatile (Groq)
   - Model ID: llama-3.1-70b-versatile
   - Used when GROQ_API_KEY is set (takes priority over OpenAI)
   - Powers: Decision simulations and AI chat

3. Built-in Mock AI Engine
   - No API key needed
   - Provides realistic demo responses out-of-the-box
   - Powers: All features in demo mode
